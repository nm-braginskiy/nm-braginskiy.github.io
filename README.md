"# nm-braginskiy.github.io

## Оптимизатор Логических Выражений

Веб-приложение для упрощения и оптимизации логических выражений с пошаговым решением.

### Возможности

- Поддержка всех основных логических операций (AND, OR, NOT, XOR, NAND, NOR, импликация, эквивалентность)
- Автоматическое упрощение выражений с применением законов булевой алгебры
- Пошаговое отображение процесса упрощения
- Подсветка синтаксиса с цветовым кодированием скобок
- **Автоматическая вставка знака умножения**: при вводе `ab` автоматически отображается как `a * b` (серым цветом)

### Синтаксис

| Символ | Операция | Пример |
|--------|----------|--------|
| `-` | NOT (отрицание) | `-a` |
| `*` | AND (конъюнкция) | `a * b` |
| `+` | OR (дизъюнкция) | `a + b` |
| `!+` | XOR | `a !+ b` |
| `!*` | NAND (Шеффер) | `a !* b` |
| `!!+` | NOR (Пирс) | `a !!+ b` |
| `->` | Импликация | `a -> b` |
| `<->` | Эквивалентность | `a <-> b` |

### Неявное умножение

Для удобства ввода знак умножения можно опускать в следующих случаях:
- `ab` → `a * b`
- `a(b+c)` → `a * (b + c)`
- `(a+b)c` → `(a + b) * c`
- `(a+b)(c+d)` → `(a + b) * (c + d)`
- `2a` → `2 * a`

Автоматически вставленные знаки умножения отображаются серым цветом для визуального различия.

### Запуск

Откройте `logic_solver/index.html` в браузере или запустите локальный сервер:

```bash
cd logic_solver
python -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере."
